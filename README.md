一、变量的定义
    使用let定义变量    let a = 12;//定义变量已经被浏览器实现

    区别：
      1. 使用var定义的变量只有函数作用域，而let定义的变量多了一个块级作用域的概念{}
      2.let不允许重复声明

    总结：其实let才是接近其他语言的变量
